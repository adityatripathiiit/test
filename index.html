DESKTOP APPLICATION

Brief Overview

We have made a desktop application on electron platform. The purpose of the application is to maintain a user friendly interface and provide a seamless connectivity across different layers of the software and hardware part.

Button-1

Button one will start the telescope server which will help in connecting the
telescope to stellarium software.

Buttons-2&3

a) Button two will connect us to Rpi which will allow us to send the data extracted from Stellarium to R-Pi.
b) Button three will start the stellarium software and will automatically start
sending the coordinates of the selected heavenly body. 

Unique Features

It is completely independent
from any other application and can be easily installed and run in any computer. We
have designed the application so that even if some script-libraries are missing, the software will automatically install them. 

Extracting Data from Stellarium

The coordinates of the celestial body are extracted from Stellarium. Stellarium is a free, open-source planetarium for your computer. We chose Stellarium, because it provides coordinates with a precision of 6 digits after the decimal point.

Conversion of the Extracted Data

The socket provides the ’Right Ascension’ and ’Declination’ angle of the selected celestial body. These angles are defined globally. We convert it into the 'azimuth and the elevation angles' of the celestial body at the particular time for the given place. The conversion requires current Universal Time Coordinated (UTC), number of days
after January 01, 2000 and, longitude and latitude value of the place.

Raspberry-Pi Application

Setting up Raspberry-Pi

We downloaded ’NOOBS’ from the Raspberry Pi website on Raspberry Pi 3 B+ model. After setting up the operating system, we downloaded the geocoder library by running the required command in ’terminal’.

R-Pi App

To create a better platform for the users, we created an interactive ’turtle window’ through python code in Raspberry Pi. After running the code and once the connection is made, the window will open automatically and allow the user to interact using the four keys - "s,t,m & r".

ELECTRICAL ASSEMBLY

The Electrical Assembly involves the following prime components:

Motors and Drivers

The stepper motors require the drivers for micro-stepping (in order to obtain better precision). We have used TB6600 stepper-motor driver along with NEMA-17 45 kg cm planetary geared motor. The connections were made accordingly.

Connecting Motors to Raspberry-Pi

We used General Purpose Input-Output (GPIO) pins of Raspberry Pi to give the signals for rotating the motors. Pins 20 and 21 were used for the first motor and Pins 19 and 26 for the second. The degree inputs of the coordinates is required to be converted to the number of steps, the motor has to rotate. The simple code to run the motors is as shown in the image.

Gears and Base

a) There are 4 gears involved: 2 for the motors and 2 for rotating the telescope along the ‘azimuth’ and ‘elevation’ planes.

b) The motors are held on the base with the help of ‘efficient screw system’ and so are the gears (which aid in rotating the telescope). 

Motor Holders

a) There are two motor holders for keeping the motors intact. One motor holder is placed on an efficient set-up build up on the telescope stand and the other is attached on the ‘base platform’ (near the rim of the telescope holder). 




Rotation Mechanism

a) HORIZONTAL: Horizontal rotation is achieved by tightening a gear with the help of connectors and ‘nut & screw system’. The gear is rotated via motor using ‘rear-reduction’ mechanism that further rotates the telescope.

b) VERTICAL: The vertical rotation was difficult as it was against the gravity. ‘Special connectors’ and ‘torque-reducing mechanisms’ were designed for this purpose.

Challenges Faced By Us

Each and every project is accompanied by a set of challenges. The prime challenges faced by us were:

Precision-Conversion Precision-Mechanical Strength of Gear

Precision - Conversion

The coordinates obtained from stellarium have 6 significant digits after decimal point. However, the conversion formula gives the coordinates having only 2 significant digits after decimal point.

Precision-Mechanical

The method of laser cutting on acrylic sheets gives a high precision but that precision is not up to the mark needed by us. This fact affected the movement of gears.

Strength of Gear

The gears manufactured through laser cutting on acryclic sheet are brittle in nature.
